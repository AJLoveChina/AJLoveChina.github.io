我们平时在开发一些demo程序，小例子的时候，经常需要在项目中引入诸如jQuery，underscore这样的开源库。

然后常见的几种做法：

  1. 从磁盘或者网络上找到相应的文件导入项目中

  2. 使用bower加载开源库， 或者你的小例子比较高端，使用npm并结合webpack一起使用

但是平常开发过程中，这都让我觉得很不方便。

有木有一个办法，让我一个命令行就找到相应库的地址，然后通过script，link标签引入这些库就好了啊？

有的！这就是bootcdn~（笔者亲自开发， 非常人性化操作， 开箱即用， 话不多说， 附上截图与命令行脚本示例）

bootcdn  jquery

那么这工具给出的cdn地址来自哪里呢？那就是 `bootcdn.cn`， 口碑比百度JS CDN要好。

GIF图给出了俩种使用方法：1）直接查询库名称，手动选择版本     2）指定库名称和版本， 模糊查询出结果

   


PS：如果有一天你忘了bootcdn如何使用，那么直接在命令行敲击，它会告诉你怎么做的

bootcdn

PPS：如果有一天你连bootcdn这个命令都忘了的时候~~ 这。。。。也有办法

npm ls -g --depth=0

注：

  1. npm ls -g 的意思是：显示npm全局安装的包

  2. --depth=0的意思是：不显示包的依赖（即dependency），如果你不指定这个参数，会发现前面一句指令输出的结果非常多， 是滴，非常非常多
