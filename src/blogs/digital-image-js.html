HTML5的canvas提供了俩个灰常强大的方法getImageData与putImageData

var canvas = document.getElementById('canvas'),
    ctx = canvas.getContext('2d'),
    arr = [];    //    保存图像像素信息的数组
arr = ctx.getImageData( 0, 0, 200, 200); //四个参数分别对应
                                        //起点横坐标，起点纵坐标，宽度，高度
// arr[0] = 第一个像素的r值
// arr[1] = 第一个像素的g值
// arr[2] = 第一个像素的b值
// arr[3] = 第一个像素的a值(a表示不透明度)
// arr[4] = 第二个像素的r值
// ...以此类推

然后我们就阔以修改数组arr中的任意一个值，也就是说明我们阔以操作一个图像中的任意一个像素的r,g,b,a值。你想做什么都阔以了！(但是web性能稍弱是一个硬伤)

可是仅仅操作了arr中的数据，并不是对图像的修改就直接生效了，我们还需要另一个牛x的方法putImageData,这个方法会把图像的数据渲染成最终的图像。

ctx.putImageData( arr, 0, 0 );    //三个参数分别是图像数据,绘制起点横坐标，纵坐标
